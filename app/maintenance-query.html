这里需对接创新部优惠券发放接口，接口文档如上。这个界面是由创新部实现的。

1. 获取可用优惠券列表

GET /api/coupons/available
Query 参数
{
  "scene": "MAINTENANCE_QUERY",  // 业务场景：报告查询
  "amount": 188,                 // 订单金额(元)，用于满减判断
  "userId": "string"             // 用户 ID(后端从 token 解)
}
响应 (按 expireAt 升序返回)
{
  "code": 0,
  "data": [
    {
      "id": "C202604001",
      "title": "报告查询满减券",
      "amount": 20,            // 面额(元)
      "threshold": 100,        // 使用门槛(元)
      "scene": "MAINTENANCE_QUERY",
      "expireAt": "2026-05-06",
      "status": "AVAILABLE"    // AVAILABLE / USED / EXPIRED
    }
  ]
}

2. 校验并锁定优惠券

POST /api/coupons/lock
Body
{
  "couponId": "C202604001",
  "orderNo":  "MQ20260429000123"
}
响应
{
  "code": 0,
  "data": { "locked": true, "discount": 20 }
}

3. 下单(含优惠券)

POST /api/maintenance-query/order
Body
{
  "vin": "LGB***********454",
  "products": ["accident", "mileage", "maintenance"],
  "payMethod": "COUPON",       // WECHAT / CARD / COUPON
  "couponId": "C202604001"     // payMethod=COUPON 时必填
}
排序与筛选
  • 列表按 expireAt 升序(最快过期的在前)。
  • 仅返回 status = AVAILABLE 且 expireAt > now 的记录。
  • 不满足 threshold 的优惠券前端置灰、不可选。
  • 距过期 ≤ 7 天的标橙色提示。
