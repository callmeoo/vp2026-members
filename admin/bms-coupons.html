优惠券 · 需求点说明 ×

本期改造：将原「运营部·优惠券」模块与新会员体系打通。

  1. 已派发优惠券：展示所有用户已领取的优惠券明细，支持按账号 / 手机 / 来源 / 状态 / 奖品活动 / 优惠券名 / 代码 / 领券时间 / 绑定车辆 多维筛选;新增「使用」操作快速核销，状态实时同步至前端卡包。
  2. 优惠券类型：维护可发放的优惠券模板。「优惠类型」字段需与「商城商品配置」中的商品类型对齐 —— 当前包含数据查询权益、保留价查询、历史成交行情、数据查询优惠券(5/10/15/20/30/50 元)、综合服务费减免券、服务中心消费券等。新增积分商城兑换型商品时，对应的优惠券类型自动出现在此列表，无需重复维护。
  3. 来源管理：「新增来源」支持配置赠送券、活动奖品、积分兑换、补偿等业务渠道;来源字段贯穿派发列表与类型列表，便于按渠道统计。
  4. 有效时长 / 使用性质 / 订单范围 / 车辆来源：四个业务字段控制优惠券生效边界:有效时长决定领券后多久内可用、使用性质限制使用环节(订单支付 / 服务费抵扣等)、订单范围限制业务场景(拍卖大厅 / 一口价 / 唯普好车等)、车辆来源限制车型品类。
  5. 导出：已派发列表支持按当前筛选条件导出 Excel，供财务对账与运营复盘。
