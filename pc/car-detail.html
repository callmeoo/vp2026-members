车辆详情 · 需求点说明 ×
  1. 查看保留价
    入口固定展示在出价区，不随车辆状态（预展中 / 竞价中 / 已成交 / 已流拍）变化。是否可查看由权益判定：① 对应等级（V2 / V3）享有此项权益的用户，点击即可查看；② V0 / V1 等级用户可消耗 50 积分对单台车进行兑换，兑换后在权益有效期内反复查看，无需再次扣减。
  2. 历史成交行情
    入口固定展示在出价区，不随车辆状态变化。是否可查看由权益判定：① V1 / V2 / V3 享有「行情查询」权益的用户，点击即可查看本车同车系历史成交记录；② V0 等级用户可消耗 100 积分对单台车进行兑换，兑换后在权益有效期内反复查看，无需再次扣减；③ 之前付费购买了带「行情查询」权益的会员卡用户，在卡片有效期内可查看。
  3. 登录拦截
    查看保留价、查看历史成交价均需登录后才可查看。未登录时移动端两个入口仍展示(PC端仅登录后可见)，点击跳转至登录页；登录后回到当前页，需再次点击该入口才会展开查看 / 兑换流程，不自动展开。
  4. 演示切换（评审专用）
    右下角浮窗以 4 行 Mock 切换组织——登录态 / 等级 / 积分兑换权益（保留价查询 ↔ 历史成交价）/ 积分场景（积分足 ↔ 积分不足）。「积分不足」会根据当前选中权益自动设为对应的不足值（保留价：30 / 历史：80），「积分足」统一设为 200，配合权益入口即可演示锁定弹窗 + 确认兑换按钮 disabled 的完整链路。其中历史行情仅车辆详情页有入口，出价页不提供。
